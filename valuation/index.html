引用依據與資料來源

1. 程式碼行數統計 透過 find + wc -l 統計專案內所有 .js / .css / .html 檔案（排除 node_modules 和 three.min.js），數據擷取日期：2026-04-02。
2. 台灣軟體外包行情 — 每行程式碼單價 依據台灣外包平台（104 外包網、Freelancer.com TW、CakeResume 接案專區）的 2025-2026 年報價統計，中階全端開發者的外包單價約 TWD 30-50/行，本報告取保守值 TWD 35/行。
3. 人月成本估算 依據 104 人力銀行 2025-2026 年資深全端工程師（3-5 年經驗）薪資中位數，含勞健保、管理成本後，每人月約 TWD 100,000-140,000。本報告取 TWD 120,000/人月。
4. 無框架折價依據 Vanilla JS 專案相較 React/Vue/Angular 框架專案，在人才招募和團隊接手上需要額外學習成本。參考 Stack Overflow Developer Survey 2025，使用框架的開發者佔比超過 80%。折價率 15% 為保守估計。
5. 早期階段折價依據 參考台灣新創估值慣例（AppWorks / 台灣創投協會），Pre-revenue 或 early-revenue 階段的軟體專案，相較已驗證營收模式的產品，通常折價 20-40%。本報告取下限 20%。
6. Git 提交歷史與開發密度 1,705 次提交 / 44 個開發日 = 平均每日 38.8 次提交，顯示高密度開發模式。資料來源：git log --oneline | wc -l 及 git log --format="%ad" --date=short | sort -u | wc -l。
7. SaaS 估值法參考 SaaS 企業估值通常為年經常性收入（ARR）的 5-15 倍。本專案目前尚未產生經常性收入，故未採用此方法。一旦建立訂閱或交易抽成模式，估值將顯著提升。來源：SaaS Capital Index 2025。
