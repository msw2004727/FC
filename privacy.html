隱私權政策

最後更新日期：2026 年 3 月 19 日

ToosterX（以下稱「本服務」）是一套運動活動報名與管理平台，透過 LINE LIFF 技術提供服務。本隱私權政策依據中華民國個人資料保護法（以下稱「個資法」）及 LINE Platform 開發者政策制定，說明我們如何蒐集、處理、利用及保護您的個人資料。

使用本服務即表示您已閱讀並同意本隱私權政策。若您不同意，請停止使用本服務。

一、資料蒐集者

本服務由 ToosterX 營運團隊負責個人資料之蒐集、處理與利用。如有任何隱私相關問題，請透過本服務 LINE 官方帳號（@830utvza）與我們聯繫。

二、蒐集之個人資料類別

依個資法第 2 條規定，本服務蒐集以下類別之個人資料：

三、蒐集之目的

依個資法第 19 條規定，本服務基於以下特定目的蒐集個人資料：

  • 活動管理：處理活動報名、候補遞補、出席管理及人數統計
  • 身份驗證：透過 LINE LIFF 登入識別使用者身份
  • 通知推播：透過 LINE Messaging API 發送活動通知、報名確認及系統公告
  • 統計分析：計算個人活動統計、排行榜、成就系統等功能
  • 服務改善：分析使用模式以改善平台功能與使用者體驗
  • 系統安全：記錄管理操作日誌以維護系統完整性與稽核需求
  • 帳號稽核：記錄登入 IP 位址與推定地理位置，用於偵測異常登入與帳號盜用風險（僅限管理員查閱）

四、資料利用方式與範圍

您的個人資料僅在上述蒐集目的之必要範圍內利用，且限於本服務平台運作所需。我們不會：

  • 將您的個人資料出售予第三方
  • 將您的個人資料用於與本服務無關之商業行銷
  • 在未經您同意之情況下公開揭露您的個人資料

五、第三方服務與資料共享

本服務使用以下第三方服務，您的資料可能依其各自隱私權政策處理：

第三方服務用途資料處理地區
Google Firebase (Firestore / Auth / Storage / Cloud Functions) 資料儲存、使用者驗證、檔案儲存、後端邏輯 Google Cloud asia-east1（台灣彰化）
LINE Platform (LIFF SDK / Messaging API) 使用者登入、個人資料取得、訊息推播 LINE Corporation 日本伺服器

上述第三方服務之資料處理均受其各自隱私權政策約束。本服務不會向上述技術基礎設施以外之第三方傳輸您的個人資料。

六、跨境傳輸

您的資料主要儲存於 Google Cloud 台灣地區機房（asia-east1），但因使用 LINE Platform 及 Cloudflare CDN，部分資料可能傳輸至日本及其他地區之伺服器。我們確保該等傳輸符合個資法第 21 條跨境傳輸之規定。

七、資料保留期間

  • 帳號資料：於您使用本服務期間持續保留；帳號刪除後於 30 日內清除
  • 活動紀錄：永久保留以維護歷史統計之完整性
  • 稽核日誌：依設定之 TTL（存留時間）自動過期刪除
  • 本機快取：儲存於您的裝置，您可隨時透過瀏覽器設定清除

八、Cookie 與本機儲存

本服務使用以下本機儲存技術：

  • localStorage：儲存登入狀態快取、使用者偏好設定
  • IndexedDB：Firebase Auth 使用 IndexedDB 維持登入 session
  • Service Worker Cache：快取靜態資源以提供離線體驗

上述儲存機制不涉及第三方追蹤 Cookie。您可透過瀏覽器設定管理或清除這些資料。

本服務並非專為未滿 14 歲之兒童設計。若我們得知已蒐集未滿 14 歲兒童之個人資料且未取得法定代理人同意，將主動刪除該等資料。法定代理人如發現此情形，請立即與我們聯繫。

十一、資料安全措施

我們採取以下措施保護您的個人資料：
